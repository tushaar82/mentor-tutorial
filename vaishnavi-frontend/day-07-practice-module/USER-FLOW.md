# User Flow for Day 7: Practice Module UI

This document describes the student's journey through the practice module with detailed flow diagrams.

## Overview

The practice module allows students to practice specific topics, receive immediate feedback, and track their mastery progress. The flow supports two modes: Learning Mode (with immediate feedback) and Test Mode (exam-like experience).

## Key User Interactions

### 1. Topic Selection
- **Action**: Student browses topics and selects one to practice
- **Result**: Navigates to configuration page

### 2. Session Configuration
- **Action**: Student configures practice session parameters
- **Result**: Session starts with specified configuration

### 3. Answering Questions (Learning Mode)
- **Action**: Student selects answer and submits
- **Result**: Immediate feedback with explanation

### 4. Answering Questions (Test Mode)
- **Action**: Student selects answer and submits
- **Result**: Automatically advances to next question

### 5. Completing Session
- **Action**: Student answers all questions
- **Result**: Session summary with mastery update

### 6. Practice Again
- **Action**: Student clicks "Practice Again"
- **Result**: New session starts with same configuration

### 7. Viewing Mastery
- **Action**: Student navigates to mastery dashboard
- **Result**: Overview of all topics with mastery levels

### 8. Adding to Review
- **Action**: Student clicks "Add to Review" on difficult question
- **Result**: Question saved to review queue for later practice

## Edge Cases Handled

1. **Page Refresh During Session**: State persists, session resumes
2. **Timer Expires**: Auto-submits session, shows summary
3. **Network Error**: Error message with retry option
4. **No Topics Available**: Empty state with message
5. **All Questions Answered Correctly**: Celebration animation
6. **Duplicate Review Questions**: Prevented by duplicate check

---

This user flow ensures students have a smooth, engaging practice experience that helps them master topics and improve their exam performance.
